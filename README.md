# matrizenergiaen

Dash Application on Energy Consumption and Energy Balance in Lisbon


## SETUP

```
git clone https://github.com/FranMacedo/matrizenergiaen.git
cd matrizenergiaen
python -m venv venv
.\venv\scripts\activate
pip install -r requirements.txt
```

## RUN

```
	python app.py
```
